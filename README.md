# GZXX 的个人博客

类似于一个模板，将文档放入对应位置并运行.ps1指令集即可直接生成所有数据并对网页进行预览。

## 前言

最近在避免完全依赖AI编写代码，Vibe Coding是一个很容易让人上瘾的东西。又想用AI学点东西，所以做了这个博客，在使用AI的同时自己编写注释，保证自己完全理解代码。毕竟以后不能说我只会用AI写代码，离开AI啥也不是了吧。

## 项目简介

GZXX_Blog 是一个使用纯前端代码实现的静态网站。

首先里面有我自己的学习笔记、技术文章和个人资料，以前我都是记在飞书上的，但是飞书客户端又卡，缓存占的又多，用网页又不方便即开即用，所以干脆直接开个仓库用markdown写了，看起来还不错。

该项目使用 HTML、CSS、JavaScript 三个最常见的前端网页构建，不需要任何额外的环境配置就能直接运行。而且代码模块化，方便个人修改和定制。

其他人也可以将自己的文章、音乐以及头像图片放在指定的文件夹里（或者自己改文件路径），即插即用。

## 如何使用

如果想要快速将该仓库改造为自己的博客，只需按照以下步骤：

1. clone/fork该仓库到本地/云端
2. 已有的所有文档、图片以及音乐，删除还是保留请随意。
3. 将你的所有文章以markdown的形式保存，并放置在articles文件夹中。（格式见下）
4. 将头像命名为head.jpg并放入data文件夹中。
5. 在profile.js中填入个人信息。（说明见下）
6. 将所有音乐文件（支持mp3、wav、ogg三种常见格式）放入musics文件夹中。
7. 在artists.js中配置艺术家名称。（说明见下）
8. 在index.html中修改网页名以及导航栏标题。
9. 在命令行中运行`.\start_blog.ps1`。
10. 保存并提交即可。之后如果选择使用github托管静态网站请看github官方文档，使用其他服务同样。
